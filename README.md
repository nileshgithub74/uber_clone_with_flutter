# Uber Clone with Flutter (Rider Side)

A mobile application clone of Uber built with Flutter, featuring real-time ride booking, Google Maps integration, and Firebase authentication for the rider side.

<img src="lib/images/ss.png" width=300 height=460>

## Features
- User authentication with Firebase
- Real-time location tracking
- Google Maps integration for ride visualization
- Ride booking functionality
- User profile management

### Prerequisites & Setup
Don't forget to insert your APIKEY for the project to work:
- **Android**: Insert your API key in `android/app/src/main/AndroidManifest.xml`
- **iOS**: Insert your API key in `ios/runner/AppDelegate.m`

### Installation
1. Ensure you have Flutter installed on your system
2. Clone this repository
3. Run `flutter pub get` to install dependencies
4. Configure your Firebase credentials
5. Run `flutter run` to launch the app

## Technologies Used
- **Flutter**: Cross-platform mobile development framework
- **Firebase**: Authentication and real-time database
- **Google Maps**: Location and route visualization
- **Provider**: State management
